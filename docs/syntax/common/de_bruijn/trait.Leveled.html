Expand description

Convert an De-Bruijn index to a De-Bruijn level

To perform this conversion, it is sufficient to track

  • the current number of De-Bruijn levels (maximum first component)
  • the current number of binders per De-Bruijn level (maximum second component for each first component). This is satisfied by the context type during typechecking.
